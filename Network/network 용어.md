# network 용어

## 네트워크

데이터 교환을 위해 전송 매체에 의해 연결된 노드들의 그룹  
(전송 매체 : 유선, 무선, 노드 : 통신이 가능한 장비)

## 호스트(host)

노드 또는 시스템, 네트워크에 연결된 컴퓨터, 라우터, 서버 등

## 프로토콜(protocol)

호스트들이 데이터 통신할 때 사용하는 통신 규칙
